# wenshen

# 该小程序模仿 `Tattoo御刺青` 小程序
# 适合学习小程序的新手熟悉小程序的开发流程
# 插件模式下的

# 该小程序最开始为插件模式下开发的所以文件目录跟小程序模式下有所差异

```
- miniprogram                   ----小程序文件目录
  - components                  ----组件库
  - images                      ----图片库
  - pages                       ----页面库
  - app.js
  - app.json
  - app.wxss
  - icon.wxss                   ----引入额外的icon的全局样式
- project.config.json           ----小程序配置文件

```
